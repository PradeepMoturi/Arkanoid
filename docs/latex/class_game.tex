\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_game}{Game}} class. 

\mbox{\hyperlink{class_game}{Game}} class creates many connections, creates the grid, handles brick ball collisions, creates new worker threads, new ball objects, starts the timer, updates the brick status, powerup status and many more..
